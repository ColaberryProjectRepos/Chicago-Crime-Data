   Create Association Between The Tables

 Follow the instructions below to create an association between the  Tables.

  1. Click on the model tab to begin creating an association.
  2. Drag the DateKey column in the ChicagoCrimeData Table to join the DateKey column in the OccurredDate table to create an association between them

  4. Drag the occurredTimeID column in the ChicagoCrimeData Table to join the ID column in the Time table to create an association between them

  6. Drag the CommunityArea column in the ChicagoCrimeData Table to join the CommAreaID column in the CommunityArea table to create an association between them

  8. Drag the DistrictID column in the CommunityArea Table to join the District column in the NewChicagoDistrict table to create an association between them

  10. Drag the DistrictID column in the District Table to join the District column in the NewChicagoDistrict table to create an association between them
